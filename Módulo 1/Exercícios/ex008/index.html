Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>.

Porém a forma correta é utilizar a tag <strong>.

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>.

Porém a forma correta é utilizar a tag <em>.

Texto Marcado

Nesta frase temos um texto marcado usando a tag <mark>.

Para mudar a cor do texto marcado é só utliizar a tag da seguinte forma: <mark style="background-color:COR;">.

Texto Grande e Pequeno

Estamos criando um Texto Grande e um Texto Pequeno nesse parágrafo.

Para fazer um texto grande usar a tag <big>, e para um texto pequeno usar a tag <small>.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Para marcar o texto como excluido, utilizar a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Para marcar o texto como inserido, utilizar a tag <ins>.

Existe também o texto sublinhado com a tag <u>.

Texto Sobrescrito

Para inserir caractéres do tipo x10 + 3, utilizamos a tag <sup>.

Texto Subscrito

Para inserir caractéres do tipo H2O, utilizamos a tag <sub>.
